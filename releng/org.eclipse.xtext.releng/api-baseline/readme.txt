How to set API Baseline.
========================

1. Open corresponding *.target file, e.g. {your_local_git_repo}/org.eclipse.xtext.releng/api-baseline/runtime_2.2.1/api-baseline.target
2. Click export icon in the right-upper corner of your target definition editor
3. Export wizard asks for a location, point him to
	 {your_local_git_repo}/org.eclipse.xtext.releng/api-baseline/runtime_2.2.1/ folder and click "Finish"
4. Open API-Baseline settings
5. Add new baseline name Xtext 2.2.1, location {your_local_git_repo}/org.eclipse.xtext.releng/api-baseline/runtime_2.2.1/
