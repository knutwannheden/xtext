How to set API Baseline.
========================

1. Open the api-baseline.target file {your_local_git_repo}/releng/org.eclipse.xtext.releng/api-baseline/api-baseline.target
2. Click export icon in the right-upper corner of your target definition editor
3. Export wizard asks for a location, point him to
	 {your_local_git_repo}/releng/org.eclipse.xtext.releng/api-baseline/xtext folder and click "Finish"
4. Open API-Baseline settings
5. Add new baseline name Xtext, location {your_local_git_repo}/releng/org.eclipse.xtext.releng/api-baseline/xtext
